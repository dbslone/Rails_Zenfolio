# ZenfolioAPI

This is a basic implementation of the Zenfolio API. More features like uploading photos will come later.

## Installation

Add this line to your application's Gemfile:

    gem 'ZenfolioAPI'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ZenfolioAPI
